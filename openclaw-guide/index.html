Command Reference

Gateway Commands

  • openclaw gateway start Start the gateway service
  • openclaw gateway stop Stop the gateway service
  • openclaw gateway restart Restart the gateway
  • openclaw status Show gateway status

Agent Commands

  • openclaw agent list List all agents
  • openclaw agent create [name] Create new agent
  • openclaw agent delete [name] Remove an agent
  • openclaw agent status [name] Check agent status

Skill Commands

  • openclaw skill list List installed skills
  • openclaw skill install [name] Install a skill
  • openclaw skill remove [name] Remove a skill

Best Practices

1. Agent Design

  • Give each agent a clear, single purpose
  • Define SOUL before building capabilities
  • Start simple, add complexity gradually

2. Memory Management

  • Use SOUL â†’ PERSONALITY â†’ MEMORY template chain

3. Agent Communication

  • Use agent-to-agent announcements
  • Define clear handoff protocols
  • Log all inter-agent communications

4. Testing

  • Test agents in isolation first
  • Use quality scoring before deployment
  • Monitor agent performance continuously
